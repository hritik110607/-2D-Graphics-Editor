# 2D Graphics Editor

A simple **2D Graphics Editor** developed in **C Programming** that allows users to create, modify, and manage basic geometric shapes on a text-based canvas. This project demonstrates fundamental concepts of **Computer Graphics**, **Data Structures**, and **Object Management** through an interactive menu-driven interface.

## Features

* Draw Rectangles
* Draw Lines (DDA Algorithm)
* Draw Circles (Midpoint Circle Algorithm)
* Draw Triangles
* Modify Existing Objects
* Delete Objects
* Display Objects on a Canvas
* Save Drawings to a Text File
* Manage Multiple Objects in a Scene

## Technologies Used

* C Programming Language
* GCC Compiler
* Standard C Libraries

  * stdio.h
  * stdlib.h
  * math.h

## Learning Outcomes

* Understanding basic computer graphics algorithms
* Working with structures and arrays in C
* Implementing object storage and management
* File handling for saving graphical output
* Developing menu-driven applications
